🎵 Music Player

📌 Description

A responsive and interactive Music Player built using HTML, CSS, and JavaScript. The application allows users to play, pause, skip, and control music with a simple and modern user interface.

✨ Features

- Play and Pause songs
- Next and Previous controls
- Progress bar
- Volume control
- Playlist support
- Album cover display
- Song title and artist name
- Responsive design
- Clean and user-friendly interface

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

🚀 How to Run

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Make sure the "songs" and "images" folders contain the required files.
4. Run the project using Live Server.
5. Enjoy the music player in your browser.
